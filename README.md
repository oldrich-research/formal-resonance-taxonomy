# Formal Resonance Taxonomy

This repository publishes the source package for:

`A Formal Taxonomy of Physical Resonance Mechanisms`

Author: Oldrich Dvorak, independent researcher.

## Contents

- `paper/formal_resonance_taxonomy.pdf` — compiled manuscript.
- `paper/main.tex` — RevTeX source.
- `paper/references.bib` — bibliography.
- `supplement/54_case_public_supplement.tsv` — 54 case-test public supplement ledger.
- `arxiv/source.tar.gz` — arXiv-style source bundle.
- `transport_manifest.md` — local publication transport manifest.
- `SHA256SUMS` — hashes for the published artifacts.

## Scope

The manuscript makes a bounded taxonomy claim over 54 documented case-tests in the public supplement. The paper appendix prints a representative 15-row subset so the five-field contract is visible in paper form. It does not claim exhaustive closure over all future resonance-like phenomena.
